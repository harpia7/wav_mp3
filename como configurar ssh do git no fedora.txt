2 - adicionar uma nova chave SSH ao ssh-agent

	iniciando ssh-agent
	
	$ eval "$(ssh-agent -s)"
	> Agent pid 59566
	
	
mais de uma chave a partir daqui repetir para cada chave...

	$ ssh-add ~/.ssh/id_ed25519_qasd137
